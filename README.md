**技术栈**: Vue3、Vue-router、Vuex、Vite、axios

### 工具🛠️

代码规范：eslint + commitlint + commitizen

自动化部署：GitHub Actions

### 使用💁‍♂️

- 安装：

  ```shell
  yarn
  ```

- 运行：

  ```shell
  yarn run dev
  ```

- 打包：

  ```shell
  yarn run build
  ```
